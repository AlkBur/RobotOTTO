# AGENTS.md

## Платформа

- ESP32C3 Dev Module
- Сборка в Arduino IDE
- Всего 4 независимых мотора, подключены через PCA9685

## Моторы

2 мотора (ноги) — сервоприводы позиционные (180°):

- POSITION: targetAngle (0–180°), optional duration (плавное движение)

2 мотора (колёса) — сервоприводы непрерывного вращения (360°):

- DRIVE: direction (FWD, BWD, STOP), speed (0–255), duration (мс)

## Архитектура

- Неблокирующая архитектура
- Не использовать `delay()`
- Использовать `millis()` или FreeRTOS
- Код оптимизирован для real-time управления

## Библиотеки

- ESP Async WebServer (ESP32Async v3.10.3)
- AsyncTCP (ESP32Async v3.4.10+)
- DFRobotDFPlayerMini
- Adafruit_PWMServoDriver
- ArduinoJson — не используется, протокол через компактные строки

## Веб-интерфейс

Два файла дублируют друг друга:

| Файл           | Назначение                                                 |
| -------------- | ---------------------------------------------------------- |
| `index_html.h` | Встраивается в прошивку (PROGMEM), используется при сборке |
| `index.html`   | Предпросмотр интерфейса в браузере без сборки              |

При изменениях интерфейса обновлять **оба** файла.

## Протокол WebSocket (ESP → Browser)

- `p05` — DFPlayer type (hex)
- `cL150,600` — Calibration left leg (min,max)
- `cR150,600` — Calibration right leg (min,max)

## Протокол WebSocket (Browser → ESP)

Компактные строки:

| Пакет         | Пример   | Описание                      |
| ------------- | -------- | ----------------------------- |
| `bX1` / `bX0` | `b31`    | Кнопка (press/release)        |
| `dU1` / `dU0` | `dL1`    | D-Pad                         |
| `tL102`       | `tR0`    | Trigger (аналоговый 0-255)    |
| `jL,127,127`  | `jL,0,0` | Joystick (x,y -128..127)      |
| `sl90`        | `sr45`   | Servo калибровка (угол 0-180) |
| `ml150`       | `xr600`  | Servo калибровка (min/max)    |
| `tl`          | `tr`     | Тест ноги (установить в 90°)  |

Коды кнопок: `X` cross, `O` circle, `S` square, `T` triangle, `1` L1, `2` R1, `3` L2, `4` R2, `Q` share, `M` options, `P` ps, `*` touch

Коды D-Pad: `U` up, `D` down, `L` left, `R` right

## План доработок

- Easing (плавное движение сервоприводов)
- Отдельный FreeRTOS task для управления моторами
- Интерполяция POSITION: завершение по достижению угла
